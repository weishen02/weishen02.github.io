使用hexo+Github Page搭建静态博客

  • Anonymous
  • 2018/09/07
  • Comments

关于搭建，主要分为两个过程，一、创建Gitbub Page仓库；二、利hexo本地编辑好，然后部署到GitHub仓库上。

创建GitHub Page

登录GitHub官网，https://github.com/,注册登录。

建立仓库（New repository），如图所示

接下来是这一界面，注意，仓库名称必须为[username].github.io，username为你自己的用户名，不是用户昵称，owner下方即为用户名。
选择默认操作，创建仓库就结束了，输入https://[username].github.io,就可以访问你的博客页面了。


安装与使用hexo

hexo是一个快速、简洁且高效的博客框架，首先利用它在本地编辑好博客，再推送到GitHub上即可。

安装hexo

安装hexo前请确认本机上已装好git与node.js,至于为什么，稍有经验的人一眼便知道了。现在开始安装hexo，打开命令行窗口，

$ npm install hexo-cli -g

本地博客服务器建立与启动

切换到自己想要存放博客内容的文件夹下，打开命令行窗口

$ hexo init <dirctory name>
$ cd <dirctory name>
$ npm install
$ hexo server

dirctory name是自己输入的名称，此时打开浏览器，输入localhost:4000，就可以访问到博客界面了。接下来这一句，

$ hexo deploy

即可将本地资源部署到Github上，Github博客也实现了更新。

作者在这里对整个过程只是做一个简答说明，关于hexo项目下各文件夹以及配置文件用途，官网上都有很详细的说明。
